# sirvivianfuchs.com

Static HTML and CSS website dedicated to the life and achievements of Sir Vivian Fuchs (1908-1999), leader of the Commonwealth Trans-Antarctic Expedition on the first land crossing of the Antarctic in 1958.

## Development

Open `index.html` in a browser. No build step required.
